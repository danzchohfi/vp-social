# Brand assets

Coloque aqui o doc de direção de marca (PDF/PPT/imagens) usado com clientes. Quando estiver no repo, Claude consegue ler e consolidar a §7 (Brand Direction) do `POSITIONING.md`.

Formatos aceitos:
- `.pdf` — extração de páginas (até 20 por leitura)
- `.png` / `.jpg` — leitura visual direta
- `.md` — transcrição manual das slides

Sugestão de estrutura quando o material chegar:

```
brand/
  direcao-de-marca-clientes.pdf       (o doc original)
  logo/
    vp-light.svg
    vp-dark.svg
  paleta.md                            (HEX, nome, uso)
  tipografia.md                        (headline + body, weights, casos)
  voz.md                               (tom, do's, don'ts, exemplos)
  mood/                                (referencias visuais)
```
